# Панель Модерации Авито (Клиент)

Фронтенд-приложение для модерации объявлений на Авито.

## Возможности

- **Список объявлений**: Просмотр всех объявлений с пагинацией, сортировкой и фильтрацией.
- **Детали объявления**: Просмотр подробной информации об объявлении, включая изображения, описание и информацию о продавце.
- **Действия модерации**: Одобрение, отклонение или запрос изменений для объявлений.
- **Статистика**: Просмотр статистики модератора и графиков (активность, решения, категории).

## Стек технологий

- **React 18**: Библиотека UI.
- **TypeScript**: Статическая типизация.
- **Vite**: Инструмент сборки.
- **Ant Design**: Библиотека UI компонентов.
- **Axios**
- **React Router DOM**: Маршрутизация.
- **Recharts**: Библиотека для графиков.

## Начало работы

1.  Установите зависимости:
    ```bash
    npm install
    ```

2.  Запустите сервер разработки:
    ```bash
    npm run dev
    ```

    Приложение будет доступно по адресу `http://localhost:5173`.

## Структура проекта

- `src/api`: API клиент и функции.
- `src/components`: Переиспользуемые компоненты (AdCard, Filters, AppLayout).
- `src/pages`: Основные страницы (AdsList, AdDetails, Stats).
- `src/types`: TypeScript интерфейсы.
- `src/utils.ts`: Вспомогательные функции.

## Комментарии

Код прокомментирован в следующих файлах:
- `src/pages/AdsList.tsx`: Логика для просмотра списка, фильтрации, пагинации.
- `src/pages/AdDetails.tsx`: Логика для просмотра деталей и действий модерации.
- `src/pages/Stats.tsx`: Логика для получения и отображения статистики.
- `src/components/Filters.tsx`: Логика формы фильтрации.
- `src/components/AdCard.tsx`: Логика отображения карточки.
- `src/api/index.ts`: API функции и типы.
